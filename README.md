# LZMODE Now Playing

Виджет для отображения текущего трека на LZMODE Radio.

- Анимированный градиентный фон
- Автообновление каждые 5 секунд
- Реклама каждые 25 секунд
- Адаптивный дизайн
- Обработка ошибок соединения

## API

GET `/current`
```json
{
  "title": "Название трека",
  "artists": "Исполнитель",
  "image": "URL обложки"
}
```
